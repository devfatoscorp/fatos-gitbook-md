# Zoom

### setZoom

Sets the zoom level of the map.

#### Parameter

| Required Parameter | Description | Type |
| :--- | :--- | :--- |
| value | Zoom level \(0-20\) | Numeric |

#### Example

```javascript
mapInstance.setZoom(5);
```

### getZoom

Gets the current zoom level of the map.

#### Return

The current zoom level of the map.

#### Example

```javascript
mapInstance.getZoom();
return 5
```
